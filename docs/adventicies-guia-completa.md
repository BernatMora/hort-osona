<h1 id="-guia-completa-de-plantes-adventícies-les-quotmales-herbesqu">🌱 Guia completa de plantes adventícies — Les &quot;males herbes&quot; que t&#x27;ajuden</h1>
<p>&gt; A l&#x27;hort ecològic <strong>no hi ha &quot;males herbes&quot;</strong>, hi ha <strong>plantes adventícies</strong>: plantes que creixen soles, sovint ajuden a l&#x27;ecosistema, i moltes són <strong>comestibles, medicinals o útils</strong>. Aquesta guia et canvia la mirada: en lloc d&#x27;arrencar-les totes, aprendràs a <strong>conviure-hi</strong>, a <strong>aprofitar-les</strong> i a <strong>gestionar-les intel·ligentment</strong>.</p>
<h2 id="1-què-són-les-plantes-adventícies">1. Què són les plantes adventícies?</h2>
<h3 id="definició">Definició</h3>
<p>Les <strong>plantes adventícies</strong> (també dites <strong>&quot;males herbes&quot;</strong> o <strong>&quot;espontànies&quot;</strong>) són plantes que <strong>creixen soles</strong> a l&#x27;hort, sense que les sembris. Inclouen:</p>
<ul>
<li><strong>Espècies natives</strong> d&#x27;Osona (margall, ortiga, dent de lleó)</li>
<li><strong>Espècies introduïdes</strong> que s&#x27;han naturalitzat</li>
<li><strong>Cultius anteriors</strong> que han rebrotat (mongeta, tomàquet)</li>
</ul>
<h3 id="per-què-apareixen">Per què apareixen</h3>
<ul>
<li><strong>Sòl descobert</strong> (la natura no tolera espais buits)</li>
<li><strong>Compactació</strong> del sòl (planta indicadora)</li>
<li><strong>Excés/defecte</strong> de nutrients</li>
<li><strong>Pertorbació</strong> (cavar, treballar el sòl)</li>
<li><strong>Llavors al sòl</strong> (banc de llavors, fins a 50.000 llavors/m²!)</li>
</ul>
<h2 id="2-la-visió-clàssica-vs-lx27ecològica">2. La visió clàssica vs. l&#x27;ecològica</h2>
<h3 id="visió-clàssica-errònia">Visió clàssica (errònia)</h3>
<ul>
<li>❌ &quot;Són dolentes, cal arrencar-les&quot;</li>
<li>❌ &quot;Competèixen amb els cultius&quot;</li>
<li>❌ &quot;Cal usar herbicides&quot;</li>
<li>❌ &quot;Són inútils&quot;</li>
</ul>
<h3 id="visió-ecològica-correcta">Visió ecològica (correcta)</h3>
<ul>
<li>✅ &quot;Són part de l&#x27;ecosistema&quot;</li>
<li>✅ &quot;Ajuden a mantenir el sòl cobert&quot;</li>
<li>✅ &quot;Atreuen pol·linitzadors&quot;</li>
<li>✅ &quot;Moltes són útils&quot;</li>
<li>✅ &quot;Són indicadores del sòl&quot;</li>
<li>✅ &quot;S&#x27;alimenten, es mediquen&quot;</li>
</ul>
<h2 id="3-per-què-les-plantes-adventícies-són-útils">3. Per què les plantes adventícies són útils</h2>
<h3 id="a-cobertura-del-sòl">A. Cobertura del sòl</h3>
<ul>
<li><strong>Protegeixen</strong> el sòl de l&#x27;erosió</li>
<li><strong>Redueixen</strong> l&#x27;evaporació</li>
<li><strong>Alimenten</strong> els microorganismes</li>
<li><strong>Creen</strong> estructura</li>
</ul>
<h3 id="b-biodiversitat">B. Biodiversitat</h3>
<ul>
<li><strong>Atreuen</strong> pol·linitzadors (flors)</li>
<li><strong>Atreuen</strong> fauna beneficiosa (insectes, ocells)</li>
<li><strong>Són</strong> aliment per a animals</li>
</ul>
<h3 id="c-indicadores-del-sòl">C. Indicadores del sòl</h3>
<p>Cada planta indica un tipus de sòl o condició:</p>
<ul>
<li><strong>Ortiga</strong> → sòl ric en N</li>
<li><strong>Margall</strong> → sòl compactat</li>
<li><strong>Dent de lleó</strong> → sòl ric</li>
<li><strong>Romaní</strong> → sòl pobre i sec</li>
</ul>
<h3 id="d-comestibles">D. Comestibles</h3>
<p>Moltes són <strong>excel·lents verdures</strong>:</p>
<ul>
<li><strong>Ortigues</strong> (sopes, truites, infusions)</li>
<li><strong>Dent de lleó</strong> (amanides)</li>
<li><strong>Cua de cavall</strong> (tisana)</li>
<li><strong>Borratja</strong> (flors, fulles)</li>
<li><strong>Romaní</strong> (condiment)</li>
</ul>
<h3 id="e-medicinals">E. Medicinals</h3>
<ul>
<li><strong>Camamilla</strong> (infusió relaxant)</li>
<li><strong>Farigola</strong> (tònica)</li>
<li><strong>Cua de cavall</strong> (cicatritzant)</li>
<li><strong>Plantatge</strong> (tos)</li>
<li><strong>Saüc</strong> (febrífug)</li>
</ul>
<h3 id="f-preparats-útils">F. Preparats útils</h3>
<ul>
<li><strong>Macerat d&#x27;ortigues</strong> (fertilitzant)</li>
<li><strong>Infusió de cua de cavall</strong> (antifúngic)</li>
<li><strong>Borratja</strong> (com a mulching viu)</li>
</ul>
<h2 id="4-les-25-plantes-adventícies-més-útils-a-osona">4. Les 25 plantes adventícies més útils a Osona</h2>
<h3 id="1-ortiga-urtica-dioica">1. ORTIGA (Urtica dioica)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 50-150 cm</li>
<li><strong>Color</strong>: verd fosc</li>
<li><strong>Fulla</strong>: ovalada, dentada, amb pèls urticants</li>
<li><strong>Flor</strong>: verdosa, penjant (maig-setembre)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Molt rica en N</strong> (perfecta per al compost)</li>
<li><strong>Macerat</strong> = fertilitzant excel·lent</li>
<li><strong>Comestible</strong> (sopa, truita, infusió)</li>
<li><strong>Medicinal</strong> (reumatisme, al·lèrgies)</li>
<li><strong>Atreu</strong> marietes i papallones</li>
<li><strong>Indicadora</strong> de sòl ric en N</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Deixar-la créixer</strong> en una cantonada</li>
<li><strong>Tallar-la</strong> 3-4 cops/any (abans de florir)</li>
<li><strong>Fer macerats</strong> amb el que tallis</li>
<li><strong>Mai florir</strong> (llavors molt invasiva)</li>
</ul>
<h3 id="2-dent-de-lleó-taraxacum-officinale">2. DENT DE LLEÓ (Taraxacum officinale)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 10-30 cm</li>
<li><strong>Color</strong>: verd brillant</li>
<li><strong>Fulla</strong>: dentada (d&#x27;aquí el nom)</li>
<li><strong>Flor</strong>: groc viu (tot l&#x27;any)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Comestible</strong> (fulles tendres, arrel torrada com a cafè)</li>
<li><strong>Medicinal</strong> (diürètica, depurativa)</li>
<li><strong>Mel·lífera</strong> (excel·lent!)</li>
<li><strong>Atreu</strong> abelles, sírfids</li>
<li><strong>Arrel</strong> profunda (porta nutrients a la superfície)</li>
<li><strong>Indicadora</strong> de sòl ric en K</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Tallar-la</strong> abans de quallar</li>
<li><strong>Menjar</strong> les fulles tendres a l&#x27;amanida</li>
<li><strong>Arrel torrada</strong>: torrar a 150°C, 30 min, moldre</li>
</ul>
<h3 id="3-margall-o-cua-de-cavall-petita-hordeum-murinum">3. MARGALL o CUA DE CAVALL PETITA (Hordeum murinum)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 30-60 cm</li>
<li><strong>Color</strong>: verd-gris</li>
<li><strong>Fulla</strong>: prima, allargada</li>
<li><strong>Flor</strong>: espigues fines amb arestes (primavera)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Indicadora</strong> de sòl compactat</li>
<li><strong>Sistemes radicals</strong> que trenquen compactació</li>
<li><strong>Mulching viu</strong> (cobertura del sòl)</li>
<li><strong>Aliment</strong> per a ocells (les llavors)</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Tallar</strong> abans de granar</li>
<li><strong>Deixar-la</strong> com a cobertura</li>
<li><strong>Cavar</strong> la zona compactada a la tardor</li>
</ul>
<h3 id="4-saüc-sambucus-nigra">4. SAÜC (Sambucus nigra)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 2-5 m (arbust)</li>
<li><strong>Color</strong>: verd fosc</li>
<li><strong>Fulla</strong>: composta, fulletes dentades</li>
<li><strong>Flor</strong>: blanques, umbel·les (maig-juny)</li>
<li><strong>Fruit</strong>: baies negres (agost-setembre)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Flor medicinal</strong> (febrífug, depuratiu)</li>
<li><strong>Fruit</strong> (melmelada, vi)</li>
<li><strong>Atreu</strong> ocells</li>
<li><strong>Arbust autòcton</strong> a Osona</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Deixar-lo</strong> créixer en una vora</li>
<li><strong>Poda</strong> anual a la tardor</li>
<li><strong>Collir flors</strong> (candy, infusions)</li>
<li><strong>Collir fruits</strong> (melmelada)</li>
</ul>
<h3 id="5-plantatge-plantago-major-p-lanceolata">5. PLANTATGE (Plantago major, P. lanceolata)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 10-30 cm</li>
<li><strong>Fulla</strong>: ovalada o lanceolada, nervis paral·lels</li>
<li><strong>Flor</strong>: espiga prima (tot l&#x27;any)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Medicinal</strong>: tos, ferides, picades</li>
<li><strong>Comestible</strong>: fulles tendres</li>
<li><strong>Indicadora</strong> de sòl calcari (P. lanceolata)</li>
<li><strong>Molt rústica</strong>, arrela fàcilment</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Deixar créixer</strong> en marges</li>
<li><strong>Collir fulles</strong> per a infusions</li>
<li><strong>Tallar</strong> si envaeix</li>
</ul>
<h3 id="6-cua-de-cavall-equisetum-arvense">6. CUA DE CAVALL (Equisetum arvense)</h3>
<h4 id="identificació">Identificació</h4>
<ul>
<li><strong>Mida</strong>: 30-50 cm</li>
<li><strong>Color</strong>: verd clar</li>
<li><strong>Tija</strong>: articulada, amb nusos (com una cua!)</li>
<li><strong>Espores</strong> al principi (a la primavera)</li>
</ul>
<h4 id="per-què-és-útil">Per què és útil</h4>
<ul>
<li><strong>Antifúngic</strong> (infusió contra míldiu, oïdi)</li>
<li><strong>Rica en sílice</strong> (bones per a les plantes)</li>
<li><strong>Arrels profundes</strong> (porten nutrients)</li>
<li><strong>Indicadora</strong> de sòl àcid i humit</li>
</ul>
<h4 id="com-gestionar-la">Com gestionar-la</h4>
<ul>
<li><strong>Fer infusió</strong> amb la planta seca</li>
<li><strong>Polvoritzar</strong> les plantes</li>
<li><strong>Controlar-la</strong> (pot ser invasiva!)</li>
</ul>

<h2 id="5-com-gestionar-les-plantes-adventícies">5. Com gestionar les plantes adventícies</h2>
<h3 id="a-principi-bàsic-gestió-no-eliminació">A. Principi bàsic: gestió, no eliminació</h3>
<pre><code>
❌ Erradicar (impossible, contaminants, innecessari)
↓ millor ↓
🌱 Gestionar amb seny
</code></pre>
<h3 id="b-estratègies">B. Estratègies</h3>
<h4 id="1-prevenció">1. Prevenció</h4>
<ul>
<li><strong>Mulching</strong> (5-10 cm) — impedeix que germinin</li>
<li><strong>Cobertura</strong> permanent del sòl</li>
<li><strong>No deixar sòl nu</strong></li>
<li><strong>Banc de llavors net</strong> (compost ben fet)</li>
</ul>
<h4 id="2-acceptació-selectiva">2. Acceptació selectiva</h4>
<ul>
<li><strong>Deixar</strong> les útils (ortiga, dent de lleó, borratja)</li>
<li><strong>Tallar</strong> abans de quallar</li>
<li><strong>Mantenir</strong> una zona &quot;natural&quot;</li>
</ul>
<h4 id="3-control-manual">3. Control manual</h4>
<ul>
<li><strong>Arrencar</strong> a mà (eina: bufador, aixada)</li>
<li><strong>Oportunitat</strong>: després de pluja</li>
<li><strong>Quan</strong>: abans de granar</li>
<li><strong>Eines</strong>: aixada, bufador, falç</li>
</ul>
<h4 id="4-cobertura">4. Cobertura</h4>
<ul>
<li><strong>Palla</strong> (5-10 cm)</li>
<li><strong>Cartó</strong> (sense cinta, sense color)</li>
<li><strong>Tela anti-adventícies</strong> (per a voreres)</li>
<li><strong>Mulching viu</strong> (plantar cobrint)</li>
</ul>
<h4 id="5-rotació">5. Rotació</h4>
<ul>
<li><strong>Alternar</strong> cultius cobreix (alfals, civada) amb altres</li>
<li><strong>Descans</strong> del sòl (amb adob verd)</li>
</ul>
<h4 id="6-solarització-per-zones-molt-infectades">6. Solarització (per zones molt infectades)</h4>
<ul>
<li><strong>Plàstic transparent</strong> a l&#x27;estiu, 4-6 setmanes</li>
<li><strong>Mata</strong> les llavors de la capa superior</li>
<li><strong>Recuperar</strong> amb compost</li>
</ul>

<h2 id="10-el-racó-quotnaturalquot-de-lx27hort">10. El racó &quot;natural&quot; de l&#x27;hort</h2>
<h3 id="què-és">Què és</h3>
<p>Una <strong>zona petita</strong> (2-5 m²) de l&#x27;hort on <strong>deixes créixer</strong> les adventícies de manera natural. Beneficis:</p>
<ul>
<li><strong>Biodiversitat</strong> augmentada</li>
<li><strong>Reserva</strong> d&#x27;insectes beneficiosos</li>
<li><strong>Font</strong> d&#x27;aliments i medicines</li>
<li><strong>Seny</strong> de salut del sòl</li>
<li><strong>Educatiu</strong> (per a nens i adults)</li>
</ul>
<h3 id="com-fer-ho">Com fer-ho</h3>
<ol>
<li><strong>Triar</strong> una cantonada de l&#x27;hort</li>
<li><strong>Marcar</strong> amb pedres o fusta</li>
<li><strong>Deixar</strong> créixer</li>
<li><strong>Gestionar</strong> les invasores</li>
<li><strong>Observar</strong> i gaudir</li>
</ul>
