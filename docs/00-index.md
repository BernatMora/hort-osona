<h1 id="-base-de-coneixement-hort-ecològic-a-osona">🌱 Base de coneixement — Hort ecològic a Osona</h1>
<h2 id="ubicació-i-context">Ubicació i context</h2>
<ul>
<li>Comarca: <strong>Osona</strong> (Catalunya central, voltant de Vic)</li>
<li>Altitud mitjana hort familiar: ~500-700 m</li>
<li>Clima: <strong>continental mediterrani</strong> — hiverns freds amb gelades (octubre–abril), estius càlids i secs (juliol–agost), pluges concentrades a primavera i tardor (~600-700 mm/any)</li>
<li>Risc: gelades tardanes (Sant Marc, 25 d&#x27;abril) i calor extrema de juliol</li>
<li>Filosofia: <strong>ecològica</strong> — sense químics de síntesi, adobs verds, rotacions, biodiversitat</li>
</ul>
