<div class="doc-hero"><img src="img/api.svg" alt="Fitxa de cultiu: Api (Apium graveolens)" loading="lazy"></div><h1 id="fitxa-de-cultiu-api-apium-graveolens">Fitxa de cultiu: Api (Apium graveolens)</h1>
<h2 id="dades-generals">Dades generals</h2>
<ul>
<li><strong>Nom científic</strong>: <em>Apium graveolens</em></li>
<li><strong>Família botànica</strong>: Umbel·líferes (Apiàcies)</li>
<li><strong>Origen</strong>: Mediterrani; conreat des de l&#x27;Edat Antiga (grecs i romans)</li>
<li><strong>Cicle</strong>: bianual (floreix el segon any)</li>
<li><strong>Profunditat de sembra</strong>: 0,5 cm (llavors molt petites)</li>
<li><strong>Germinació</strong>: 15-25 dies (llent!)</li>
</ul>
<h2 id="varietats-tradicionals-catalanes">Varietats tradicionals catalanes</h2>
<ul>
<li><strong>Api d&#x27;Espinavell</strong> (Pirineu, resistent al fred)</li>
<li><strong>Api ple</strong> (penes amples, molt productiu)</li>
<li><strong>Api d&#x27;Aranjuez</strong> (penes fines, tradicional)</li>
<li><strong>Api de Brunsvic</strong> (penes gruixudes, modern)</li>
</ul>
<h2 id="sembra">Sembra</h2>
<ul>
<li><strong>Per planter</strong>: febrer-març (amb calor, 18-20°C)</li>
<li><strong>Trasplantament</strong>: maig-juny, quan té 4-5 fulles</li>
<li><strong>Marc</strong>: 30×30 cm (api de penes) o 25×25 cm (api arrel)</li>
<li><strong>A sol</strong>: a partir de l&#x27;abril, amb molta humitat</li>
</ul>
<h2 id="cultiu">Cultiu</h2>
<ul>
<li><strong>Necessitats d&#x27;aigua</strong>: MOLT altes (no tolera la sequera); reg constant</li>
<li><strong>Sol</strong>: semi-ombra (millor que ple sol a l&#x27;estiu a Osona)</li>
<li><strong>Sòl</strong>: profund, fèrtil, amb molta matèria orgànica</li>
<li><strong>Temperatura òptima</strong>: 15-21°C</li>
<li><strong>Associacions bones</strong>: porro, ceba, tomàquet, cols</li>
<li><strong>Associacions dolentes</strong>: pastanaga (cosina) i gira-sol</li>
</ul>
<h2 id="collita">Collita</h2>
<ul>
<li><strong>A partir de</strong>: 4-5 mesos després de sembrar</li>
<li><strong>Com</strong>: tallar les penes exteriors (continuar creixent) o arrencar la planta sencera</li>
<li><strong>Producció</strong>: 1-2 kg/m²</li>
<li><strong>Conservació</strong>: nevera fins a 3 setmanes; es pot assecar o congelar</li>
</ul>
<h2 id="plagues-i-malalties">Plagues i malalties</h2>
<ul>
<li><strong>Septòria</strong> (taca foliar): varietats resistents, evitar humitat a les fulles</li>
<li><strong>Mosca de l&#x27;api</strong> (<em>Philophylla heraclei</em>): mines a les fulles</li>
<li><strong>Cucs grisos</strong>: plagues de plàntules</li>
<li><strong>Mildiu</strong>: varietats resistents</li>
</ul>
<h2 id="notes-dx27osona">Notes d&#x27;Osona</h2>
<ul>
<li>A Osona, sembrar a l&#x27;abril pot ser massa aviat si l&#x27;hivern s&#x27;allarga</li>
<li>L&#x27;api aguanta bé les gelades suaus (fins a -5°C)</li>
<li>Els apis de penes prefereixen terres fondes; els d&#x27;arrel, terres més aviat soltes</li>
<li>A la tardor, cobrir amb palla per allargar la collita</li>
</ul>
