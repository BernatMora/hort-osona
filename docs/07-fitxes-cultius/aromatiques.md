<h1 id="-aromàtiques-a-lx27hort-dx27osona">🌿 Aromàtiques a l&#x27;hort d&#x27;Osona</h1>
<h2 id="per-què-plantar-aromàtiques">Per què plantar aromàtiques?</h2>
<ul>
<li><strong>Repel·leixen plagues</strong> amb les seves olors (més potents que molts insecticides!)</li>
<li><strong>Atrauen pol·linitzadors</strong> (abelles, sírfids, borinots, papallones)</li>
<li><strong>Acoloreixen l&#x27;hort</strong> tot l&#x27;any</li>
<li><strong>Cuina</strong>: milloren tots els plats (i els conserves)</li>
<li><strong>Medicinals</strong>: infusions, ungüents, tintures</li>
<li><strong>Sòl</strong>: algunes (alfàbrega, anet) milloren el gust del tomàquet veí</li>
<li><strong>Marges</strong>: defineixen i protegeixen l&#x27;hort</li>
</ul>

<h2 id="disposició-recomanada-a-lx27hort">Disposició recomanada a l&#x27;hort</h2>
<h3 id="-la-quotbarrera-aromàticaquot-al-voltant-de-lx27hort-tot-el-">🌺 La &quot;barrera aromàtica&quot; (al voltant de l&#x27;hort, tot el perímetre)</h3>
<ul>
<li><strong>Mínim 2-3 espècies diferents</strong> barrejades</li>
<li>Recomanat: <strong>romaní + sàlvia + farigola</strong> (o menta en test, aïllada)</li>
<li>Avantatge: confon les plagues amb olors variades</li>
</ul>
<h3 id="-el-quotracó-dx27aromàtiquesquot-2-3-m²-en-un-cantó">🪴 El &quot;racó d&#x27;aromàtiques&quot; (2-3 m² en un cantó)</h3>
<ul>
<li>Totes les perennes juntes</li>
<li>Plantes d&#x27;exigències similars (sol, poc reg, sòl pobre)</li>
<li>A prop de la cuina per collir fàcilment</li>
</ul>
<h3 id="-disperses-entre-els-cultius">🌱 Disperses entre els cultius</h3>
<ul>
<li><strong>Alfàbrega al peu de cada tomàquet</strong> (1-2 alfàbregues per tomàquet)</li>
<li><strong>Orenga al peu de pebrots i tomàquets</strong></li>
<li><strong>Cibulet entre les pastanagues</strong></li>
<li><strong>Caputxina a les vores</strong> (flors comestibles!)</li>
<li><strong>Calèndula al llarg del perímetre</strong></li>
<li><strong>Anet prop dels cogombres i pastanagues en flor</strong></li>
</ul>

<h2 id="com-assecar-aromàtiques-per-guardar-tot-lx27any">Com assecar aromàtiques (per guardar tot l&#x27;any)</h2>
<ol>
<li><strong>Collir</strong> al matí, quan la rosada s&#x27;ha eixugat</li>
<li><strong>Rentar</strong> i eixugar bé</li>
<li><strong>Penjar</strong> en manats en lloc ventilat, fosc i sec</li>
<li><strong>Esperar 2-3 setmanes</strong></li>
<li><strong>Triturar</strong> i guardar en pots de vidre amb tapa</li>
<li><strong>Durada</strong>: 1 any (després perden aroma)</li>
</ul>
<h2 id="per-a-infusions-recepta-bàsica">Per a infusions (recepta bàsica)</h2>
<ul>
<li><strong>Digestiva</strong>: menta + farigola + camamilla</li>
<li><strong>Relaxant</strong>: melissa + flor de taronger + til·la</li>
<li><strong>Tònica</strong>: romaní + sàlvia + orenga</li>
<li><strong>Respiratòria</strong>: eucaliptus + farigola + espígol + anet</li>
</ul>
