<h2 id="-context">🌍 Context</h2>
<ul>
<li><strong>Comarca</strong>: Osona (Catalunya central)</li>
<li><strong>Altitud</strong>: ~500-700 m</li>
<li><strong>Clima</strong>: continental mediterrani — hiverns freds amb gelades (octubre–abril), estius càlids i secs (juliol–agost), pluges concentrades a primavera i tardor (~600-700 mm/any)</li>
<li><strong>Filosofia</strong>: ecològica — sense químics de síntesi, adobs verds, rotacions, biodiversitat</li>
</ul>
<h2 id="-com-treballar">🛠️ Com treballar</h2>
<p>Cada fitxa segueix una estructura comuna: dades generals, calendari a Osona, associacions, sòl, sembra, conreu, plagues, collita i varietats locals.</p>
<p>Hi ha 3 versions de cada tema:</p>
<ul>
<li><code>.md</code> — font original editada amb markdown</li>
<li><code>-imprimible.html</code> — versió HTML preparada per imprimir</li>
<li><code>-2026.pdf</code> — PDF generat (a partir de l&#x27;HTML)</li>
</ul>
<h2 id="-sincronització-entre-ordinadors">💻 Sincronització entre ordinadors</h2>
<p>Aquest repositori es treballa des de dos ordinadors (Mac personal + Windows de la feina) sincronitzats amb <strong>Git + GitHub</strong>.</p>
<p>Comandes bàsiques del dia a dia:</p>
<pre><code>
git pull                                          # baixar canvis de l&#x27;altre PC
git add . &amp;&amp; git commit -m &quot;descripció&quot; &amp;&amp; git push   # pujar els meus canvis
</code></pre>
<p>Per a la instal·lació al Windows, consulta <a href="SETUP-WINDOWS.md" target="_blank" rel="noopener">SETUP-WINDOWS.md</a>.</p>

<h2 id="-lloc-web-unificat">🌐 Lloc web unificat</h2>
<p>Els documents seleccionats al catàleg són accessibles des d&#x27;un sol portal amb sidebar,</p>
<p>cercador i hash routing. Generat per <code>site/build.py</code> a partir de <code>template.html</code>.</p>
<pre><code>
python site/build.py          # regenera site/index.html
python -m http.server 8765    # servidor local
# obre http://127.0.0.1:8765/site/index.html
</code></pre>
<p>Més detalls a <a href="SETUP-SITE.md" target="_blank" rel="noopener">SETUP-SITE.md</a>.</p>
