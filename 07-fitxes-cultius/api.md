# Fitxa de cultiu: Api (Apium graveolens)

## Dades generals
- **Nom científic**: *Apium graveolens*
- **Família botànica**: Umbel·líferes (Apiàcies)
- **Origen**: Mediterrani; conreat des de l'Edat Antiga (grecs i romans)
- **Cicle**: bianual (floreix el segon any)
- **Profunditat de sembra**: 0,5 cm (llavors molt petites)
- **Germinació**: 15-25 dies (llent!)

## Varietats tradicionals catalanes
- **Api d'Espinavell** (Pirineu, resistent al fred)
- **Api ple** (penes amples, molt productiu)
- **Api d'Aranjuez** (penes fines, tradicional)
- **Api de Brunsvic** (penes gruixudes, modern)

## Sembra
- **Per planter**: febrer-març (amb calor, 18-20°C)
- **Trasplantament**: maig-juny, quan té 4-5 fulles
- **Marc**: 30×30 cm (api de penes) o 25×25 cm (api arrel)
- **A sol**: a partir de l'abril, amb molta humitat

## Cultiu
- **Necessitats d'aigua**: MOLT altes (no tolera la sequera); reg constant
- **Sol**: semi-ombra (millor que ple sol a l'estiu a Osona)
- **Sòl**: profund, fèrtil, amb molta matèria orgànica
- **Temperatura òptima**: 15-21°C
- **Associacions bones**: porro, ceba, tomàquet, cols
- **Associacions dolentes**: pastanaga (cosina) i gira-sol

## Collita
- **A partir de**: 4-5 mesos després de sembrar
- **Com**: tallar les penes exteriors (continuar creixent) o arrencar la planta sencera
- **Producció**: 1-2 kg/m²
- **Conservació**: nevera fins a 3 setmanes; es pot assecar o congelar

## Plagues i malalties
- **Septòria** (taca foliar): varietats resistents, evitar humitat a les fulles
- **Mosca de l'api** (*Philophylla heraclei*): mines a les fulles
- **Cucs grisos**: plagues de plàntules
- **Mildiu**: varietats resistents

## Notes d'Osona
- A Osona, sembrar a l'abril pot ser massa aviat si l'hivern s'allarga
- L'api aguanta bé les gelades suaus (fins a -5°C)
- Els apis de penes prefereixen terres fondes; els d'arrel, terres més aviat soltes
- A la tardor, cobrir amb palla per allargar la collita
