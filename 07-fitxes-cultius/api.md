# Fitxa de cultiu: Api (Apium graveolens)

## Dades generals
- **Família**: Apiàcies (Umbel·líferes)
- **Origen**: Mediterrani (conreat des de fa 3.000 anys; l'avantpassat era l'api de mar, amarg)
- **Cicle**: biennal (any 1: tija i fulles; any 2: flor i llavor)
- **Profunditat d'arrel**: 30-40 cm (superficials i horitzontals)
- **Exigència de nutrients**: **alta** (menjador fort)
- **Necessitats hídriques**: **altes** (no tolera la sequera)
- **Rusticitat a Osona**: bona. Resisteix gelades (fins -5°C), però necessita cobertura
- **Tolerància a gelades**: **mitjana** (les varietats verdes resisteixen més que les blanques)

## Per què és bona per a l'hort?
- **Tija cruixent i refrescant**, ideal per a sopes, amanides, sucs verds
- **Poc problemàtic** amb plagues (l'olor repel·leix algunes plagues de col)
- **Bon company** de tomàquet, col, porro, ceba
- **Aprofitament màxim** (tija, fulles i llavor)
- **Llavor pròpia fàcil** d'obtenir al segon any

## Tipus principals
- **Api verd (de fulla)**: tiges primes, gust intens, molt rústic. Varietat "Tall Utah"
- **Api blanc (Dulce)**: tiges gruixudes, blanques, tendres. Clàssic per a sopes
- **Api rosat**: varietats italianes, tiges amb tonalitat rosada
- **Api fulla (per a sopa)**: tipus "Par-Cel", més fàcil de cultivar a Osona

### Varietats adaptades a Osona
- **"Tall Utah 52-70"**: api verd, rústic, bona mida
- **"Golden Self-Blanching"**: api que es blanqueja sol
- **"Par-Cel"** (api de fulla): el més fàcil per a horts petits
- **"Plein Blanc Doré"**: api blanc clàssic, per blanquejar

## Calendari a Osona

| Període | Activitat | Notes |
|---|---|---|
| **Febrer-març** | Sembrar planter (a cobert, 18-20°C) | L'api és lent: 8-10 setmanes |
| **Abril-maig** | Trasplantar a terra (20-25 cm entre plantes) | Quan el sòl >10°C |
| **Maig-juny** | Primeres tiges tendres | Collir individualment |
| **Juliol-agost** | Reg abundant + blanquejar (api blanc) | Embolicar amb paper o cartó |
| **Setembre-novembre** | Collita principal | A mesura que es necessita |
| **Octubre-novembre** | Protegir amb túnel o palla | Per allargar la collita |
| **Abril** | Florir (si no s'ha collit) | Per obtenir llavor |

### A Osona, el calendari ideal
- **Api d'estiu** (collita agost-octubre): sembrar febrer, trasplantar abril
- **Api tardor** (collita novembre-gener): sembrar març, trasplantar maig
- **Api fulla** (tot l'any): sembrar cada 2-3 mesos, collir fulles tendres

## Associacions
- **Bones**: tomàquet, col, porro, ceba, all, julivert, espinac
- **Dolentes**: pastanaga (competència, i atreu les mateixes plagues), api silvestre
- **Acompanya bé**: cols (l'api repel·leix la mosca de la col)

## Sòl i preparació
- **pH ideal**: 6.0-7.0 (tolera un pH àcid, però prefereix neutre)
- **Tipus de sòl**: profund, fresc, ric en matèria orgànica, ben drenat
- **Preparació**: llaurar a fons (30-40 cm), incorporar 4-5 kg/m² de compost madur
- **Adobat previ**: fons amb compost a la tardor; durant el creixement, aplicacions d'ortiga líquida quinzenals

## Sembra
- **Profunditat de sembra**: 0.5 cm (gairebé superficial, necessita llum per germinar)
- **Marc de plantació**: 20-25 cm entre plantes × 30-40 cm entre files
- **Densitat**: ~10-16 plantes/m²
- **Tècnica de sembra**: en safates amb substrat fi; cobrir lleugerament i mantenir humit
- **Truc**: la llavor és molt petita — barrejar amb sorra fina per sembrar uniforme
- **Truc 2**: les llavors fresques (1-2 anys) germinen millor

## Conreu
- **Reg**: abundant i constant (25-30 mm/setmana). Mai deixar assecar el sòl
- **Encoixinat**: molt recomanat amb palla o restes de poda (5-8 cm) — manté la humitat
- **Tutors**: no cal
- **Poda / esporgat / despuntat**: tallar tiges externes primer (recol·lecció progressiva), deixar el cor

### Blanquejat (per a api blanc)
- 3 setmanes abans de la collita, emboliciar les tiges amb paper de diari, cartó o tires de plàstic blanc
- O apilar terra al voltant de la base
- Mantenir 3 setmanes → tiges blanques i tendres

## Plagues i malalties
- **Septòria (Septoria apiicola)**: fong que causa taques grogues a les fulles. **Prevenció**: rotació de 3-4 anys, evitar mullar les fulles en regar
- **Mosca de la pastanaga**: atreta per l'olor — pot afectar l'api, però menys
- **Pugó**: en brots nous, tractar amb sabó potàssic
- **Escarabat de l'api**: en alguns anys, fulles rosegades; tractar amb B. thuringiensis
- **Virosi**: planta mustia i grogosa — eliminar immediatament

## Collita
- **Quan**: les tiges externes tenen 2-3 cm d'amplada (pot anar de 4-5 mesos a 8-10 mesos)
- **Com**: tallar les tiges externes a 2-3 cm de la base, deixant el cor perquè rebroti
- **Rendiment esperat**: 3-5 kg/m² (tiges externes)
- **Conservació**: en nevera 2-3 setmanes; congelat (tallat i blanchat 2 min), fins 6 mesos
- **Truc**: les fulles s'assequen i serveixen com a condiment (com el julivert sec)

## Curiositats
- L'api és considerat un **"aliment negatiu" en calories** (en cremar més digestió que no pas aporta)
- A l'**Edat Mitjana**, es creia que les llavors d'api eren **afrodisíaques**
- A l'**antic Egipte**, l'api es col·locava a les tombes dels faraons
- A la **Grècia clàssica**, els guanyadors dels Jocs Nemeus rebien una corona d'api
- La **varietat "Par-Cel"** és un híbrid entre api de fulla i api de tija, desenvolupat a Itàlia

## Llavor pròpia
- Deixar 3-4 plantes al terreny el segon any
- A la primavera emetran una tija floral alta (fins 1.5 m)
- Les llavors maduren a mitjan estiu (juliol-agost)
- Recollir les umbel·les quan es tornen marrons
- Les llavors es conserven 3-4 anys
- **Atenció**: l'api es creua amb api silvestre (api bord) — aïllar 1 km
