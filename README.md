## 🌍 Context

- **Comarca**: Osona (Catalunya central)
- **Altitud**: ~500-700 m
- **Clima**: continental mediterrani — hiverns freds amb gelades (octubre–abril), estius càlids i secs (juliol–agost), pluges concentrades a primavera i tardor (~600-700 mm/any)
- **Filosofia**: ecològica — sense químics de síntesi, adobs verds, rotacions, biodiversitat

## 🛠️ Com treballar

Cada fitxa segueix una estructura comuna: dades generals, calendari a Osona, associacions, sòl, sembra, conreu, plagues, collita i varietats locals.

Hi ha 3 versions de cada tema:
- `.md` — font original editada amb markdown
- `-imprimible.html` — versió HTML preparada per imprimir
- `-2026.pdf` — PDF generat (a partir de l'HTML)

## 💻 Sincronització entre ordinadors

Aquest repositori es treballa des de dos ordinadors (Mac personal + Windows de la feina) sincronitzats amb **Git + GitHub**.

Comandes bàsiques del dia a dia:
```bash
git pull                                          # baixar canvis de l'altre PC
git add . && git commit -m "descripció" && git push   # pujar els meus canvis
```

Per a la instal·lació al Windows, consulta [SETUP-WINDOWS.md](SETUP-WINDOWS.md).

## 🌐 Lloc web unificat

Els documents seleccionats al catàleg són accessibles des d'un sol portal amb sidebar,
cercador i hash routing. Generat per `site/build.py` a partir de `template.html`.

```bash
python site/build.py          # regenera site/index.html
python -m http.server 8765    # servidor local
# obre http://127.0.0.1:8765/site/index.html
```

Més detalls a [SETUP-SITE.md](SETUP-SITE.md).
