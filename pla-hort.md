# 🗺️ Plànol de l'hort — 20 m² (4×5 m)

> **Hort d'Osona, filosofia ecològica**
> Aplicable aquesta temporada (2026) i base per a la **rotació 2027**.

## Principis del disseny
1. **Sol**: ideal orientació sud/sud-est. Les plantes altes al nord, les baixes al sud (per no fer ombra).
2. **Ceba + Pastanaga** molt a prop (associació clàssica antimosca).
3. **Ceba allunyada de la mongeta** (oposar-les a l'hort).
4. **Cucurbitàcies** (carbassa + carabassó) en un extrem — ocupen molt i s'expandeixen.
5. **Aromàtiques perennes** al perímetre (romaní, sàlvia, farigola, orenga).
6. **Acompanyants** (alfàbrega, calèndula, caputxina) intercalades.
7. **Camins** de 40-50 cm per accedir sense trepitjar cultius.
8. **Compostador i galliner** fora de l'hort però a prop.

## Opció A — Hort clàssic en **4 parcel·les rectangulars (4×1,25 m cadascuna) + 2 camins**

```
                    NORD (més ombra, plantes altes)
  ┌──────────────────────────────────────────────────────────┐
  │  AROMÀTIQUES: romaní, sàlvia, farigola, orenga           │ 0,5 m
  │  (tot el perímetre nord, 4 plantes)                       │
  ├──────────────────────────────────────────────────────────┤
  │  PARCEL·LA 1 (4×1,25 m = 5 m²)                            │
  │  🫘 MONGETA (mata baixa, 4 fileres)                       │
  │  (solars) + alfàbrega entremig                            │
  ├──────────────────────────────────────────────────────────┤
  │  PARCEL·LA 2 (4×1,25 m = 5 m²)                            │
  │  🍅 TOMÀQUETS (3 plantes per filera, amb tutors)          │
  │  Alfàbrega al peu, clavell de moro, orenga a la vora      │
  ├──────────────────────────────────────────────────────────┤
  │  PARCEL·LA 3 (4×1,25 m = 5 m²)                            │
  │  🫑 PEBROTS (3 fileres, amb encoixinat)                   │
  │  Ceba com a vora (1 filera ceba al voltant pebrot)        │
  ├──────────────────────────────────────────────────────────┤
  │  PARCEL·LA 4 (4×1,25 m = 5 m²)                            │
  │  🧅 CEBES (4 fileres) + 🥕 PASTANAGUES (4 fileres,        │
  │  alternades, per repel·lir la mosca)                      │
  │  + 🥬 ENCIAMS (2 fileres al costat)                       │
  ├──────────────────────────────────────────────────────────┤
  │  PARCEL·LA 5 (4×1,25 m = 5 m²) — opcional o al costat    │
  │  🎃 CARBASSES + 🥒 CARABASSONS + blat de moro             │
  │  (3 germanes: carbassa + mongeta + blat de moro)          │
  │  Aquestes envairan, millor en un extrem                   │
  ├──────────────────────────────────────────────────────────┤
  │  CAMINS DE 40 cm entre parcel·les                          │
  │  Palla o grava als camins                                  │
  └──────────────────────────────────────────────────────────┘
                    SUD (més sol, plantes baixes)

  ❌ ERROR A EVITAR: posar les cebes (parcel·la 4) a prop de la
     mongeta (parcel·la 1). Aquí estan a 2 m, perfecte.
```

### Avantatges
- **4 parcel·les clares** → perfecte per a la **rotació de 4 anys**
- **Ceba + pastanaga** a la mateixa parcel·la (associació clàssica)
- **Ceba allunyada de la mongeta** (3 m de distància)
- **Plantes altes (tomàquet, pebrot) al centre**; cucurbitàcies que envaeixen al sud o al costat
- **Aromàtiques al perímetre** que repel·leixen i atrauen

```
  ┌────────────────────────────────────────────────────────┐
  │ NORD                                                    │
  │  AROMÀTIQUES (tot el perímetre, 1-2 m d'amplada)        │
  │                                                         │
  │  F1: 🌽 BLAT DE MORO (1,2 m d'alçada) ─────────────    │
  │  F2: 🫘 MONGETA (entutors del blat de moro)             │
  │  F3: 🥒 CARABASSONS (sòl cobert)                        │
  │  F4: 🎃 CARBASSES (al final, que envaeixin fora)        │
  │  F5: 🍅 TOMÀQUETS (amb tutors, 1,8 m alçada)            │
  │  F6: 🫑 PEBROTS (60 cm alçada)                          │
  │  F7: 🧅 CEBES (al peu dels pebrots!)                    │
  │  F8: 🥕 PASTANAGUES (al costat de les cebes)            │
  │  F9: 🥬 ENCIAMS (múltiples sembres)                     │
  │  F10: 🧅 CEBES D'HIVERN (tardor)                        │
  │                                                         │
  │  CAMINS: 50 cm entre fileres                             │
  │  Caputxina + calèndula entremig                         │
  │                                                         │
  │  SUD                                                     │
```

### Avantatges
- **Més senzill de plantar** (filera a filera)
- **Associacions clares**: mongeta amb blat de moro, ceba amb pebrot i pastanaga
- **Ceba allunyada de la mongeta** (estan a 3 fileres de distància)

## Opció C — Hort en **bancal profund (1,2 m ample × 4 m llarg × 3 bancals)**

```
  ┌────────────────────────────────────────────────────────┐
  │  BANCAL 1 (1,2×4 = 4,8 m²) — Solanàcies                │
  │  🍅 Tomàquets (6-8 plantes) + 🫑 Pebrots (4-5)         │
  │  Alfàbrega intercalada, orenga al cap                   │
  ├────────────────────────────────────────────────────────┤
  │  BANCAL 2 (1,2×4 = 4,8 m²) — Arrels i bulbs            │
  │  🧅 CEBES (4 fileres) + 🥕 PASTANAGUES (4 fileres)     │
  │  + 🥬 ENCIAMS (testos al peu)                           │
  │  Caputxina a la vora                                    │
  ├────────────────────────────────────────────────────────┤
  │  BANCAL 3 (1,2×4 = 4,8 m²) — Cucurbitàcies i llegum.   │
  │  🎃 CARBASSES + 🥒 CARABASSONS + 🌽 BLAT DE MORO        │
  │  + 🫘 MONGETA DE CANYA                                   │
  │  (3 germanes + mongeta)                                 │
  ├────────────────────────────────────────────────────────┤
  │  CAMINS: 50-60 cm entre bancals                         │
  │  PERÍMETRE: aromàtiques perennes                        │
  │  TESTOS: alfàbrega, menta, enciam a la cuina            │
  └────────────────────────────────────────────────────────┘
```

### Avantatges
- **Bancals permanents** (no trepitjar, millora el sòl)
- **Rotació de 3 anys** (solana/arrel/cucurbita) fàcil de recordar
- **Ceba i mongeta separades** (a 2,5 m, perfecte)
- **Sembres esglaonades** més fàcils dins de cada bancal

---

## Recomanació final

Per a 20 m² a Osona, **et recomano l'Opció C (bancals profunds)**. Per què?

1. **20 m² és petit**: bancals estrets (1,2 m) et permeten treballar **sense trepitjar** el sòl (mai més de 1,2 m d'ample)
2. **El sòl a Osona tendeix a compactar-se** (continental, pluja forta): els bancals elevats ho eviten
3. **Rotació de 3 anys** (no de 4-5-6): més senzilla per a 8 cultius
4. **Ceba i mongeta perfectament separades** (2,5 m de distància)
5. **Cucurbitàcies aïllades** en un bancal (no envairan la resta)

## On col·locar el compostador i el galliner

```
  ┌─────────────────────────────────────────────────────┐
  │  HORT (bancals)                                     │
  │   ─────────────                                     │
  │  compostador    galliner                            │
  │  (1,5×1,5 m)     (mínim 4 m²)                      │
  │  ↓                                                  │
  │  bidons d'aigua de pluja                            │
  │                                                     │
  │  Distàncies:                                        │
  │  - Compostador → hort: 3-5 m                        │
  │  - Galliner → hort: 4-5 m (olor)                    │
  │  - Galliner → compostador: 1-2 m (per abocar fems)  │
  │  - Bassa → hort: 1-2 m                              │
  └─────────────────────────────────────────────────────┘
```

## Acompanyants a intercal·lar (regla d'or)

| Cultiu | Acompanyant | On |
|---|---|---|
| **Tomàquet** | Alfàbrega (1-2 peus), clavell de moro, orenga, calèndula | Al peu i a la vora |
| **Pebrot** | Ceba (filera), alfàbrega, orenga | Vora |
| **Mongeta** | Blat de moro, carabassó, carbassa | Mates/próxim |
| **Enciam** | Rave, all, cibulet | Entremig |
| **Pastanaga** | Ceba (intercalada), porro, romaní | Filera alterna |
| **Carbassa** | Mongeta, blat de moro | Bancal mixt |
| **Carabassó** | Mongeta, blat de moro, rave, caputxina | Bancal mixt |
| **Ceba** | Pastanaga, camamilla, pebrot | A la vora |

## A evitar

- ❌ **Ceba + Mongeta** (oposar-les)
- ❌ **Ceba + cols** (que vindran a la tardor: plantar-les en un altre lloc)
- ❌ **Fonoll** a prop de gairebé tot (aïllar-lo)
- ❌ **Sàlvia** a prop del cogombre (si n'afegeixes algun dia)
- ❌ **Menta** al sòl (només en test, és invasiva)
